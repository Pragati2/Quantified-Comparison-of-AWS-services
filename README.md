This project aimed to explore, analyze, and build a machine learning algorithmusing R and SageMaker. The Random Forest Classifier provided us with an accuracy of 84% while the logistic regression provided 63%, and SGD classifier with 59%. Hence, the
Random Forest model appears to be a better option for such data. One of the drawbacks ofthe data is the past data. We might use a recent data frame that has been updated in future.
These new data points can then be used to make predictions or train new models for more accurate results. With larger dataset tuning parameters, we may try to improve the model's
accuracy.
